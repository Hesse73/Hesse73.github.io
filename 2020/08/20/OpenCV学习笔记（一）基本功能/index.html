一、图像载入与显示

读取图像

使用函数cv2.imread() :

img = cv2.imread(filepath, flag)

filepath：要读入图片的完整路径.

flag：读入图片的标志 ：

  • cv2.IMREAD_COLOR：默认参数，读入一副彩色图片，忽略alpha通道（即不使用透明度），其值为1.
  • cv2.IMREAD_GRAYSCALE：读入灰度图片，其值为0.
  • cv2.IMREAD_UNCHANGED：读入完整图片，包括alpha通道，其值为-1.

注：若图像路径设置错误，并不会报错，此时img的值为None

显示图像

使用函数cv2.imshow() :

cv2.imshow(WindowName, img)  # 将图片img显示在WindowName指定的窗口中
cv2.waitKey(0)
'''
waitKey()是一个键盘绑定函数，函数参数单位为毫秒，即在指定毫秒数内等待是否有键盘输入，
若有则返回输入值，否则返回-1
由于计算机系统有延迟，故实际等待的时间应为大于指定的时间

如果参数为0，则程序会一直停止直到任意一个键输入并返回其值
'''
cv2.destroyAllWindows()  # 清除所有窗口

若cv2.imshow() 中指定的窗口不存在，则会自动生成一个名为WindowName的窗口，也可以使用cv2.namedWindow(name,flag) 来手动生成窗口.

清除窗口时还可以用cv2.destroyWindow(WindowName), 来清楚指定窗口.

保存图像

使用函数cv2.imwrite()：

cv2.imwrite(filepath, img)  # 将图像img以PNG格式保存在filepath位置

下面给出一个完整实例：

将图像以灰度图格式载入，显示之，若输键盘入’s‘ 则保存并退出，或按’ESC‘ 直接退出不保存.

import cv2

img = cv2.imread('test.jpg', 0)
cv2.imshow('image', img)
key = cv2.waitKey(0)
if key == 27:  # 27为ESC键的值，输入为ESC则直接退出
    cv2.destroyAllWindows()
elif key == ord('s'):  # 输入为's'键则保存并退出
    cv2.imwrite('test.png', img)
    cv2.destroyAllWindows()

注意：在某些机器上返回的键盘值不是ASCII编码的，可能需要将 key = cv2.waitKey(0) 改成key = cv2.waitKey(0) && 0xFF，从而使key只取最后一个字节

二、视频载入与显示

从本地载入视频

为了播放视频，需要使用VideoCapture对象，其参数可以为视频文件的路径或者摄像头的编号. 

import cv2

cap = cv2.VideoCapture(0)

while(cap.isOpened()):  # 当获取的视频不为空时
    ret, frame = cap.read()  # 读取一帧画面，返回两个参数，ret记录获取情况是否异常，frame记录图像该帧图像

    flip = cv2.flip(frame, 1)  # 翻转图像，第二个参数为0表示x轴，1表示y轴
    cv2.imshow('video', flip)  # 显示y轴翻转后的图像
    if cv2.waitKey(25) == 27:
        break

cap.release()  # 释放
cv2.destroyAllWindows()

注意：此处为逐帧读取并逐帧显示视频，因此waitKey()所等待的时间将会决定视频播放的速度。例如对于24帧的视频，即24帧每秒，考虑到计算机本身的运行时间，一般需设置等待20~40ms.

从摄像头获取视频

类似上述方法，只需将：

cap = cv2.VideoCapture('video.avi')

改为：

cap = cv2.VideoCapture(0)  # 一般计算机只有一个摄像头，编号可输入0或-1

保存视频

保存视频时，需要使用VideoWriter对象，并使用VideoWriter_fourcc函数编码FourCC格式

VideoWriter初始化格式为：

VideoWriter(filename, fourcc, fps, frameSize[, isColor])
  • 第一个参数为文件路径
  • fourcc指定编码格式
  • fps为帧率
  • frameSize为画面尺寸
  • isColor指定黑白或彩色

VideoWriter_fourcc()使用方法为：

fourcc = VideoWriter_fourcc('M', 'J', 'P', 'G')
#或者使用：
fourcc = VideoWriter_fourcc(*'MJPG')

这里并不建议使用OpenCV处理视频，最好使用FFmpeg等工具，下面给出实例：

import cv2

cap = cv2.VideoCapture(0)

#设置fourcc 并创建VideoWriter 对象
fourcc = cv2.VideoWriter_fourcc(*'XVID')
save = cv2.VideoWriter('save.avi', fourcc, 24, (640, 480), True)

while(cap.isOpened()):
    ret, frame = cap.read()
    if ret == True:
        frame = cv2.flip(frame, 1)
        #保存此帧图像
        save.write(frame)

        cv2.imshow('cap', frame)
        if cv2.waitKey(1) & 0xFF == 27:
            break
    else:
        break

cap.release()
save.release()
cv2.destroyAllWindows()

注意：若要保存为黑白视频，除了需要将VideoWriter() 中isColor 参数设置为False 外，还需要将存储的图像转化为灰度图，即：

frame = cv2.flip(frame, 1)
#转换颜色为灰度图
frame = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
#保存此帧黑白图像
save.write(frame)

最后保存的即为黑白视频
